## Dockerized Media Server

This script allows to create an automated media server using docker containers.

The currently added containers are 
### UI
- qBittorrent
- Jellyfin
- Heimdall
- Filebrowser
- Jellyseerr
### Indexers
- Jackett
- Sonarr
- Radarr
- Bazarr

<b>Installation guide can be found in the documentation.<b>
